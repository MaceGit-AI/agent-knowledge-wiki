# addons/ — optional add-ons (NOT wiki content)

This folder holds **optional, self-contained add-ons** for the knowledge wiki —
visualizations and helpers that *read* the wiki but are **not** part of its
knowledge. Nothing here is indexed as content; delete any add-on and the wiki is
unaffected.

(Internal maintenance scripts that enforce invariants live in `scripts/`
— `wiki-audit.py`, `gen-adapters.py`. Add-ons here are user-facing features.)

## wiki-graph

`wiki-graph/wiki-graph.py` renders the wiki as an **interactive graph**: nodes = pages,
edges = `[[wikilinks]]`, click a node → its `**Summary**`. Output `wiki-graph.html` is
**git-ignored** and regenerated by the post-commit hook.

Run manually:
```
python addons/wiki-graph/wiki-graph.py "<wiki-root>"
```
then open `addons/wiki-graph/wiki-graph.html` in a browser.
